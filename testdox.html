Leilao (Domain\Entities\Leilao)

  • Leilao nao deve receber lances repetidos
  • Leilao nao deve aceitar mais de 5 lance por usuario
  • Leilao deve receber lances with data set "Teste-Com-2-Lances"
  • Leilao deve receber lances with data set "Teste-Com-1-Lances"

Avaliador (Infra\Service\Avaliador)

  • Deve trazer maior lance with data set "Teste-Em-Ordem-Crescente"
  • Deve trazer maior lance with data set "Teste-Em-Ordem-Decrescente"
  • Deve trazer maior lance with data set "Teste-Em-Ordem-Aleatória"
  • Deve trazer menor lance with data set "Teste-Em-Ordem-Crescente"
  • Deve trazer menor lance with data set "Teste-Em-Ordem-Decrescente"
  • Deve trazer menor lance with data set "Teste-Em-Ordem-Aleatória"
  • Avaliador deve buscar os 3 maiores valores with data set "Teste-Em-Ordem-Crescente"
  • Avaliador deve buscar os 3 maiores valores with data set "Teste-Em-Ordem-Decrescente"
  • Avaliador deve buscar os 3 maiores valores with data set "Teste-Em-Ordem-Aleatória"
